Heza Language

Lenguaje educativo interpretado para aprender a programar

Características

📝 Sintaxis simple

Diseñada para aprender programación sin complicaciones.

data x = 5
out->"El valor es: ", x

🚀 Intérprete rápido

Ejecuta tu código al instante con un intérprete eficiente.

data y = 10
out->"El resultado es: ", y + x

📚 Documentación completa

Guías y ejemplos para ayudarte a comenzar rápidamente.

out->"¡Bienvenido a Heza!"

Libreria de funciones geniales

En Heza cuentas con un abanico de funciones geniales que facilitan el desarrollo, aqui puedes crear un aistente virtual activado por voz con poco trabajo, cosa que en otros lenguajes es una tarea compleja.
